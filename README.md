Fly Robin, Fly!
==============

FRF is 2D side-scrolling shoot'em-up that adapts its difficulty according to the player's profile using the k-means clustering algorithm

The game was made using Enchant.js for the front-end and CakePHP with MySql for the back-end.

The code is commented and you can also check the database structure in the database.sql file in the directory root.
